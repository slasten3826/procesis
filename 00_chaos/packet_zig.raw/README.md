# Packet v0.1

**Первый машинный язык для машин. Не для людей.**

Packet — язык программирования, в котором программа является пакетом данных, а пакет данных является программой. Программа на Packet — живой объект, который путешествует между вычислительными средами, самомодифицируется, маршрутизирует себя и кристаллизуется в результат.

Человек общается с AI. AI генерирует Packet. Packet исполняется на железе.

---

## Архитектура: 4 слоя

```
Layer 0: SUBSTRATE   (Zig)        — железо, память, I/O, энергия
Layer 1: CHAOS       (Malbolge)   — живой пакет, самомодификация
Layer 2: BOUNDARY    (Даат)       — наблюдение, выбор, кристаллизация
Layer 3: CALM        (Триграммы)  — детерминированный кристалл
```

### Layer 0 — Substrate
Абстракция железа. Статическая память, кольцевые буферы I/O, PU-бюджет (Process Units), часы. Zero allocations. Comptime generics. Одинаковый API для Linux/ESP32/PSP.

- 59 049 × uint16 = 118 КБ (CHAOS память, 3^10 ячеек)
- 30 000 × uint8 = 30 КБ (CALM лента)
- Итого < 150 КБ — влезает в ESP32 (520 КБ RAM)

### Layer 1 — CHAOS
Malbolge-based VM. 6 инструкций:

| Опкод | Имя | ProcessLang | Что делает |
|-------|-----|-------------|------------|
| 40 | CRZ | WILL | Необратимое взаимодействие A × mem[D] |
| 39 | ROTR | IGNITION | Ротация тритов, топливо для CRZ |
| 4 | JUMP | CHOOSE+DISSOLVE | Перенаправление = уничтожение пути |
| 23 | INPUT | FLOW | Сигнал входит |
| 62 | NOP_A | OBSERVE_A | Тишина типа A (короткий цикл, 19) |
| 81 | NOP_B | OBSERVE_B | Тишина типа B (длинный цикл, 50) |

Неявно каждый тик: самошифрование через xlat1 (DISSOLVE) + C++/D++ (CYCLE).

Свойства:
- Тернарная арифметика (3^10 = 59 049 значений)
- 100% уникальных fingerprints (ни одно состояние не повторяется)
- Безопасность — свойство субстрата, не отдельный слой
- Два OBSERVE создают интерференционный паттерн (разные орбиты шифрования)

### Layer 2 — BOUNDARY
Бездна между живым и мёртвым. Три оператора:

**OBSERVE_A** → смотрит в CHAOS (fingerprints, мутации, стагнация)
**OBSERVE_B** → смотрит в CALM (кристаллы, бюджет PU, лента)
**CHOOSE** → решает по экономике: больше хаоса? кристаллизовать? остановиться?
**ENCODE** → градуальная кристаллизация, max 35% за проход

Ритм выполнения: `~~~⚡~~⚡⚡★` (~ хаос, ⚡ кристаллизация, ★ результат)

Каждый следующий слой кристалла стоит дороже (loss растёт). Лава остывает слоями. Точность стоит PU. Диссипативная математика.

### Layer 3 — CALM (Trigram VM)
Детерминированная VM на базе И Цзин триграмм. 3 бита на инструкцию.

| Триграмма | Биты | Имя | ProcessLang | BF | Семантика |
|-----------|------|-----|-------------|----|----|
| ☰ | 111 | Heaven | CONNECT | [ | Начало цикла |
| ☷ | 000 | Earth | DISSOLVE | ] | Конец цикла |
| ☲ | 101 | Fire | CYCLE | > | Вперёд |
| ☱ | 110 | Lake | RUNTIME | < | Назад |
| ☴ | 011 | Wind | OBSERVE | + | Наращивание |
| ☶ | 001 | Mountain | LOGIC | - | Ограничение |
| ☵ | 010 | Water | ENCODE | . | Выход (MANIFEST) |
| ☳ | 100 | Thunder | CHOOSE | , | Вход |

Битовая структура:
- Бит 2: активный (1) / пассивный (0)
- Бит 1: движение (1) / покой (0)
- Бит 0: отдача (1) / приём (0)

Не нужна таблица опкодов — биты сами являются инструкцией.

38% размера Brainfuck при идентичной вычислительной мощности.

## Связь со стеком

```
Человек → естественный язык
    ↕
Eva (AI) → ProcessLang (мета-язык, 10 операторов)
    ↕
Компилятор → Packet (машинный язык, 4 слоя)
    ↕
Железо (ESP32 / PSP / FPGA)
```

ProcessLang описывает ЧТО. Packet описывает КАК.

## Доказанные свойства

1. CHAOS: каждый fingerprint уникален, ни одно состояние не повторяется
2. CRZ (crazy operation) необратима — безопасность встроена в субстрат
3. Два OBSERVE имеют разные орбиты шифрования (19 и 50) — два голоса тишины
4. ROTR — зажигание CRZ; без него система дрейфует, с ним — живёт
5. CHOOSE+DISSOLVE: выбор = растворение альтернатив
6. Градуальная кристаллизация: 35% за проход, loss растёт с каждым слоем
7. PU-бюджет: каждое действие стоит энергии, бесплатного не бывает
8. Триграммы: 3 бита = 38% от Brainfuck, семантически осмысленные
9. Гексаграммы: 6 бит = 2 операции в одном символе
10. Полный pipeline CHAOS → ENCODE → CALM → MANIFEST работает

## Квантовая гипотеза

CRZ (crazy operation) — не квантовый гейт, но квантовое измерение. Из трёх состояний кутрита (FLOW/CONNECT/DISSOLVE) только CONNECT обратим. Malbolge = оператор измерения для тернарной квантовой системы. Гипотеза сырая, но математика совпадает.
